# My Translator  
AI-Powered Desktop Translation App

My Translator is a fast and smart desktop application that allows you to translate text between **any language or dialect** using **AI technology**.  
No need to copy–paste text into a browser anymore — just **select**, **copy**, and **translate instantly**.

---

## Features

- **Instant Translation** using AI  
- Supports *all languages & dialects*  
- Translate by **just copying text**  
- Smooth & easy-to-use Windows Forms UI  
- Uses an **external AI API** (Cohere AI Key required)  

---

## How It Works

1. Select any text  
2. Copy it  
3. Press **The Shortcut**  
4. My Translator instantly shows the translation  

No switching apps, no extra steps.

## Important notes
1. The project does not contain database files. If you want to download the database files, you will find them at this link: [Database](https://www.dropbox.com/scl/fo/blln221kfatrl581m5h6b/AAi0nNkQqaX9SVXjOXDv7K4?rlkey=6hx7q5uq05ioxcjllnyt3jsk4&st=73k2ljjo&dl=0).
2. Put the database in \bin\DB or modify the connection string and place it wherever you want.
3. If you only want to try the program, you can download the setup file and install the program on your device with LocalDB. It will work directly (LocalDB is included with the setup file, you won't need to download it separately). You can download the setup file from this link: [Setup File](https://www.dropbox.com/scl/fo/i5gwq26qcshpvmdwu4ihp/AC6qmw4ZiSYz49UaAZlo4T4?rlkey=fshxyes7wcap0u43ydbdhgok7&st=z8x8suq2&dl=0).

##### Don't forget to read the instructions from the "How To Use My Translator" window. The instructions are short and will take less than two minutes to read, but they are very important to be able to use the program correctly.
